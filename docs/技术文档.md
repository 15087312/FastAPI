# FastAPI 库存微服务 - 技术文档

## 📋 项目概述

一个专业的、生产级的库存微服务，能够在多平台、高并发环境下安全管理库存，防止超卖，支持微服务化架构。

**技术栈版本**:
- Python 3.12+
- FastAPI 0.104+
- PostgreSQL 15+
- Redis 7+
- SQLAlchemy 2.0+
- Pydantic 2.5+

### 核心特性
- ✅ **防超卖机制**：Redis Lua 脚本原子操作，Kafka 异步同步数据库
- ✅ **高性能缓存**：Redis 缓存层加速读取操作，QPS 突破 1000+
- ✅ **三层调用架构**：API / Celery / CLI 多种调用方式
- ✅ **完整审计日志**：所有操作都有详细日志记录（AOP 切面）
- ✅ **幂等性保证**：通过 Redis 实现请求去重
- ✅ **优雅降级**：Redis 故障时自动降级到只读模式
- ✅ **完整健康检查**：数据库、Redis、连接池、系统资源全方位监控
- ✅ **布隆过滤器**：快速判断商品 ID 是否存在，减少无效数据库查询

## 🔧 核心组件说明

### 1. InventoryService（库存服务）

**当前实现**: `app/services/inventory_reservation.py`

```python
class InventoryReservationService:
    def __init__(self, db: Session, cache_service: InventoryCacheService = None)
    
    # 读操作（带缓存）
    def get_product_stock(self, warehouse_id: str, product_id: int) -> int
    def batch_get_stocks(self, warehouse_id: str, product_ids: List[int]) -> dict
    
    # 写操作（以数据库为准）
    def reserve_stock(self, warehouse_id: str, product_id: int, quantity: int, order_id: str) -> bool
    def confirm_stock(self, order_id: str) -> bool
    def release_stock(self, order_id: str) -> bool
    
    # 维护操作
    def cleanup_expired_reservations(self, batch_size: int = 500) -> int
```

**关键实现**:
- ✅ 纯 Redis 操作，Lua 脚本保证原子性
- ✅ Kafka 异步同步数据库，不阻塞主流程
- ✅ Redis 失败时优雅降级
- ✅ 异步事件循环兼容（get_running_loop）

### 2. 缓存策略

**实现**: `app/services/inventory_cache.py`

- **读缓存**: Cache-Aside 模式
  - 优先查缓存，未命中则返回（无数据库回退）
  - key 格式：`stock:available:{warehouse_id}:{product_id}`
  - TTL: 永不过期
- **写操作**: Redis Lua 脚本原子执行
  - 预占：检查 → 扣减 → 记录，三步原子完成
  - 批量操作：使用 Redis pipeline 优化性能
- **数据同步**: Kafka 异步写入数据库
  - 不阻塞主流程
  - 最终一致性保证

**缓存预热**:
- 应用启动时批量加载热点数据
- 按仓库分批加载，避免一次性加载过多数据

### 3. 数据库设计（Kafka 异步写入）

**实现**: `app/models/product_stocks.py`

**关键字段**:
```python
class ProductStock(Base):
    id = Column(BigInteger, primary_key=True, autoincrement=True)
    warehouse_id = Column(String(32), nullable=False, index=True)
    product_id = Column(BigInteger, ForeignKey("products.id"), nullable=False)
    available_stock = Column(Integer, nullable=False, server_default="0")
    reserved_stock = Column(Integer, nullable=False, server_default="0")
    frozen_stock = Column(Integer, nullable=False, server_default="0")
    safety_stock = Column(Integer, nullable=False, server_default="0")
```

**约束**:
- 唯一索引：`(warehouse_id, product_id)`
- CHECK 约束：`available_stock >= 0`
- 外键约束：引用 `products.id`

**数据同步方式**:

### 4. 健康检查

**实现**: `app/main.py`

**检查项**:
1. **数据库连接**: 执行 `SELECT 1`
2. **Redis 连接**: 执行 `PING`
3. **连接池状态**: size, checked_in, checked_out, overflow
4. **系统资源**: CPU, memory (使用 psutil)
5. **Kafka 消费者**: 运行状态

**响应示例**:
```json
{
  "status": "healthy",
  "checks": {
    "database": "ok",
    "redis": "ok",
    "db_pool": {"size": 11, "checked_out": 2, "overflow": 0},
    "system": {"cpu_percent": 27.4, "memory_percent": 86.0}
  }
}
```

## 🚀 部署指南

```bash

cd FastAPI_mall

# 2. 创建环境变量
cp .env.example .env
# 编辑 .env 文件配置

# 3. 启动基础服务
```
